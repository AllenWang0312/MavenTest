
### how to import 
#### add  repository to your build.gradle(project)
   
   
   ```groovy
  allprojects {
        repositories {
            maven { url "https://raw.githubusercontent.com/OoswantoO/MavenTest/master" }
        }
    }
``` 

#### import like this in you app

  ```groovy
 dependencies {
        
        implementation 'com.github.OoswantoO:kbaselib:1.1.2'
   
    }
```
